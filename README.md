# GOAT Arbitrage

Dealer sourcing engine for finding profitable sealed card inventory to resell on eBay.

## Core assumptions
- eBay fee: 12.5%
- Avg shipping cost: $7
- Must account for demand/velocity, not just profit
- Slow-moving inventory is usually a pass

## Source sites
- Dave & Adam's
- Steel City Collectibles
- Blowout Cards
- Midwest Cards
- CardGiants

## Goal
Paste a deal-page URL and return a ranked Buy/Pass report based on eBay comps, profit, ROI, and sell-through.
